# Introduction
This is a project which collects Liyuu's photographs on [Bilibili](www.bilibili.com).  
It also can be used for other UP's albums.
# Dependences
1. selectnium
2. tqdm
3. wget
4. numpy
5. opencv-python
6. pytorch
7. facenet-pytorch  
# Use
config your path to chromedriver in main.py
python3 main.py  

### Part 1
Crawler - webdriver
### Part 2
Face recognition - FaceNet(MTCNN)
